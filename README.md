### Get Started

```sh

# jspmをグローバルインストール
npm install -g jspm

git clone https://github.com/MSakamaki/gulpJSPM
cd gulpJSPM

npm install
jspm install

```

### es6 transpiler use switching

```sh
jspm dl-loader --6to5
# or
jspm dl-loader --traceur
```

### gulp tasks

##### use command

```sh
gulp help
```
 + 説明は`gulp/taskDescription.json`に書く
